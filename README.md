# PRODUCT REQUIREMENTS DOCUMENT (PRD)
## WildGuard AI - Wildlife Early Warning System Website

## 1. Executive Summary
### 1.1 Product Vision
A centralized dashboard for forest rangers and authorities to:
- Receive real-time wildlife alerts from LoRa-connected camera traps
- View animal detection history with images
- Monitor ranger deployment status
- Manage alert zones (2-zone system)

## 2. User Stories
- **Ranger:** View real-time alerts, see location on map, acknowledge deployment, view animal images, update status.
- **Officer:** Analyze alert history (7 days), filter alerts, manage camera locations, export data.
- **Operator:** Monitor system health, trigger test alerts, view live feeds.

## 3. Functional Requirements
- **Dashboard:** Live alert feed, interactive map view, system status panel.
- **Alerts Management:** History table, detailed alert view (images/metadata), bulk image upload/classification.
- **Zone Management:** Camera configuration and zone mapping.
- **Ranger Management:** Roster and deployment tracking.
- **Reports:** Analytics charts and report export.
